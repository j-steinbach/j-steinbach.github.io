# KISS

Keep it simple, stupid.

I +like+ love to run before I walk. Which means I want my [[jekyll]] blog to

- be stylable through TailwindCSS 
  >> https://jekyll.ohsostatic.com/devops/how-to-use-tailwind-css-with-jekyll-on-github-pages
- connect and interoperate with Obsidian and Wikilinks, and also show a map that connects all my notes
  >> https://refinedmind.co/obsidian-jekyll-workflow
  >> https://maximevaillancourt.com/blog/setting-up-your-own-digital-garden-with-jekyll
- allow for insertion of bibliographic entries and citations 
  >> https://github.com/inukshuk/jekyll-scholar
- show my drawings/pictures and sync them with Instagram (possibly)
- host my literate configs and sync them with Github
- do an activity overview, ...
- have an awesome styling, support font ligatures, ..
- have templates for blogs, add frontmatter

So let's take it slow and first get this going, then host and publish it on Github and then add my writings. Simple, stupid, great.
